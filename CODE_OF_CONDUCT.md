# Code of Conduct

This project follows the Contributor Covenant 2.1 standard in practical terms:

- Use welcoming and respectful language.
- Respect different viewpoints and experience levels.
- Give and receive constructive feedback professionally.
- Avoid harassment, public or private abuse, and disclosure of private information.

Maintainers may remove comments, issues, or contributions that make the project unsafe or unproductive for contributors.

## Enforcement

Report conduct concerns to the repository maintainer through GitHub. Until a project-specific contact address is configured, use the maintainer profile linked from the repository owner. Maintainers will review reports privately, act proportionally, and prioritize contributor safety.

Reference: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
