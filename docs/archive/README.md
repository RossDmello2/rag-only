# Archive

This folder preserves historical source for traceability. Files here are not loaded by `index.html` and are not maintained as runtime code.

`script_fixed.legacy.js` predates the current static-app boundary. It may contain legacy UI/provider paths, including client-side cloud-key handling patterns that are intentionally unsupported in the current app. Do not copy archived code into the runtime without a fresh security review, feature document update, and tests.
