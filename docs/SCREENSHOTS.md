# Screenshots

Maintained screenshots live in `docs/assets/screenshots/`.

| File | What it shows |
| --- | --- |
| `home.png` | Assistant panel, local service status, welcome state, composer |
| `main-workflow.png` | Sanitized sample TXT upload and mocked answer flow |
| `settings.png` | Settings modal with local service, model, chunking, and retrieval controls |
| `mobile.png` | Mobile viewport layout |

These screenshots are real browser captures of the current app UI. They were captured with Playwright against a local static server. External Ollama and Qdrant responses were mocked with sanitized sample data so no private local files, service data, tokens, or production URLs appear in the images.

No generated image is represented as a product screenshot.
